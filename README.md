**如何看论文：**

找一篇顶会/刊。

看完后，去看它引用的文献，再去看那些论文引用了它。

看背景，存在的问题，需要解决什么问题。

看introduce，再自行思考 问题 如何解决。

然后再向下看。

看它是如何发现。

背景介绍：提出问题 存在的缺陷。

introduce 后 先自行思考 问题 方案。

再向下看 他是怎么发现问题的，如何解决问题的，用到了什么方法。
