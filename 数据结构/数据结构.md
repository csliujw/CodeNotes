# 基础知识

## 位运算

- 如何用位运算得到最低位：x & ((~x)+1) = x 的最低位

  ```shell
  # 假设x是 0111
  ~x = 1000
  ~x + 1 = 1001
  x&(~x)+1 = 1000 & 1001 = 0001 # 与操作，相同才为1
  ```

- 如何用位运算得到最低位：x & (-x) = x 的最低位

  - -x = ~x + 1
