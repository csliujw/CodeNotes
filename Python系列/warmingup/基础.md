# Pymysql

- %s 占位符，避免 SQL 注入

- wsl 访问 windows 

  - windows 需要关闭防火墙

  - mysql 需要设置允许远程访问

    ```sh
    # 用Windows登录MySQL，然后执行以下语句
    use mysql;
    update user set host = '%' where user = 'root';
    FLUSH PRIVILEGES; # 必须执行这一步，否则无效
    ```

  - python 端需要安装 cryptography

- cursor 可以用 with（上下文管理器）自动释放持有的资源
