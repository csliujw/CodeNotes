# Day03

- bool 隐式转换

  - 数值类型：0 为 False
  - 字符串：空字符串为 False
  - None：None 为 False
  - 列表：[] 为 False

- 短路操作：

  - 从左往右判断，如果遇到操作数为假（隐式判断），则返回该操作数；如果都为真，则返回最后一个数（最右边的）

    ```python
    print(0 and 666) # 0
    print(88 and 0 and 666) # 0
    print(888 and 666) # 666
    print(1100 and 666) # 666
    ```
